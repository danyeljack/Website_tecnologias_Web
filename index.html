HTML - HyperText Markup Language

HTML é a linguagem de marcação utilizada para criar a estrutura básica de páginas web. Ele define os elementos da página como cabeçalhos, parágrafos, links, imagens, entre outros.

CSS - Cascading Style Sheets

CSS é a linguagem de estilo usada para definir a aparência e o layout das páginas web. Com o CSS, podemos alterar cores, fontes, tamanhos, posicionamentos e muito mais.

JavaScript

JavaScript é uma linguagem de programação que permite a criação de funcionalidades interativas nas páginas web, como validação de formulários, animações e interações dinâmicas.

PHP (Hypertext Preprocessor)

PHP é uma linguagem de script do lado servidor usada para criar sites dinâmicos e interativos. Ela é bastante usada para gerenciar bases de dados e gerar conteúdo dinâmico em tempo real.
